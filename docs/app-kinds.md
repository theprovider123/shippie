# App Kinds Are Retired

Shippie's public taxonomy is no longer Local / Connected / Cloud.

The public app kind is now:

> Local Tool

Capabilities describe what the tool can do:

- works offline
- secure backup
- reference data used
- local AI
- private relay via Shippie
- shares with my tools
- local database
- local files

Legacy kind detection still exists internally because it helps migrate older apps and explain historical deploy reports. It should not be used as new public product language.

The authoritative policy is now [docs/strategy/local-tools-policy.md](strategy/local-tools-policy.md).
